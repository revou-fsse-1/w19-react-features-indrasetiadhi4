## Installation

```bash
# Clone the repository
$ git clone https://github.com/revou-fsse-1/w19-react-features-indrasetiadhi4.git

# Change directory
$ cd w19-react-features-indrasetiadhi4

# Install Dependencies
$ pnpm install

# Run app
$ pnpm run dev
```
